# composer
+ Dependency Manager for PHP
+ like node: npm, yarn
+ like ruby: bundler
+ Structure:
```
vendor/
	various libraries
composer.json  [ package_list ]
composer.lock  [ strict version restriction ]
```

# words
+ PSR: PHP Standard Recommendation
+ PFIG: PHP Framework Interop Group
+ composer: Dependency Manager for PHP
